## Live site

- [MelbourneCoinBoard](https://melbournecoinboard.vercel.app/)


### About

This is a side project, I created on my spare time, for study purpose about Web3 and cryptocurrency.

I use react.js, material-ui, redux-toolkit, chart.js, html, css for frontend and call Coinranking/api/ to get cryptocurrency data.
